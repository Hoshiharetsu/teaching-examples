# Factorial Using Recursion (Java Teaching Example)

## Overview
This example demonstrates how to compute the factorial of a number using recursion. It introduces recursive problem-solving by breaking a problem into smaller instances of itself.

---

## Teaching Focus

This example helps students understand:

- Recursion
- Base cases and recursive cases
- Call stack behavior
- Data type limitations and overflow

---

## Instructional Approach

The program introduces the factorial definition and implements:

- Base case: return 1 when n ≤ 0
- Recursive case: n × factorial(n - 1)

Students are encouraged to experiment with different input values and data types.

---

## Key Concepts Demonstrated

- Recursion
- Function calls
- Mathematical modeling
- Data type limitations

---

## Why This Matters

Recursion is a foundational concept used in many algorithms, including tree traversal and divide-and-conquer strategies.
