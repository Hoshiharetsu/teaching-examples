# Modular Shift Cipher (Java Teaching Example)

## Overview
This example introduces modular arithmetic through the implementation of a shift cipher. The program encodes a user-provided message by shifting each character in the alphabet by a specified number of positions.

---

## Teaching Focus

This example helps students understand:

- Modular arithmetic
- Character manipulation (ASCII/Unicode)
- Loop-based string processing
- Handling negative values in modulo operations
- Refactoring complex expressions

---

## Instructional Approach

The program evolves through stages:

1. Implementing a direct modular arithmetic solution
2. Refactoring into intermediate steps for clarity
3. Discussing edge cases such as negative shifts

---

## Key Concepts Demonstrated

- Modulo operator (`%`)
- Character-to-integer conversion
- String traversal
- Method design

---

## Why This Matters

This example connects mathematical concepts to programming and highlights differences in how languages handle modulo operations.
