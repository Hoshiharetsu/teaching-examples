# FizzBuzz Teaching Example (Java)

## Overview
This example demonstrates how to introduce the classic FizzBuzz problem in an instructional setting. Rather than simply presenting a solution, the program is structured to guide students through the problem-solving process.

---

## Teaching Focus

This example helps students understand:

- Iteration using loops
- Conditional logic
- The modulus operator (%)
- Problem decomposition
- Refactoring logic into methods

---

## Instructional Approach

Students are guided through key questions:

- How do we iterate from 1 to 100?
- How do we determine if a number is divisible by 3 or 5?
- How can we structure the logic clearly?

The final solution separates logic into a helper method for readability and reuse.

---

## Key Concepts Demonstrated

- `for` loops
- `if / else if / else` conditionals
- Modulus operator (`%`)
- Method decomposition

---

## Why This Matters

FizzBuzz is a simple but effective problem for reinforcing fundamental programming concepts and logical thinking.
